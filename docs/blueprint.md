# **App Name**: CEO Journey

## Core Features:

- Weekly Progress System: The central 'Next Week' button that processes weekly changes like age, hunger, finances (income/expenses) and triggers events.
- Dynamic Character Status Display: A UI to visualize current character stats such as Age, Mood, Health, Hunger levels, and monetary status (cash, bank balance).
- Essential Resource Management: Players can purchase food to manage hunger and track basic recurring expenses like housing rent. Effective management of these resources can earn players 'Resource Management Proficiency' points, which in turn can lead to increased efficiency, reduced costs, or new economic opportunities. Mismanagement directly impacts character status and gameplay progression.
- Basic Career & Education Path: A simplified education menu (e.g., 'Work Directly', 'Attend S1 University') and a job board for entry-level positions, enabling basic career progression and income generation.
- AI-Driven Weekly Events & News: A generative AI tool to create varied and context-aware random events (e.g., job offers, minor illnesses, unexpected expenses, news headlines) each week, requiring player decisions and influencing market conditions for the Investment System.
- Game State Persistence: Functionality to save and load game progress, allowing players to resume their simulation across sessions.
- Advanced Investment System: Players can invest in various assets like stocks, cryptocurrencies, and commodities, tracking market fluctuations and potential returns or losses, influenced by weekly news and events.

## Style Guidelines:

- Primary color: A deep, authoritative blue (#1055CC) for a sense of corporate stability and focused growth.
- Background color: A very light, desaturated blue (#EEF1F6) to maintain a clean, minimalist, and easy-to-read dashboard feel.
- Accent color: A vibrant, clear cyan (#2ED5FF) for interactive elements and key indicators, providing a fresh contrast.
- Body and headline font: 'Inter', a modern sans-serif for excellent readability across all UI elements and data displays.
- Simple, clean, and intuitive linear icons that complement the minimalist interface and ensure clear communication of actions and statuses.
- Dashboard-like layout for the main screen, featuring key character statistics clearly displayed as bars or numbers, with distinct navigational menus.
- Subtle, fluid transitions and micro-animations for feedback on user interactions and state changes, avoiding anything distracting.
